# RadHAR：基于毫米波雷达的人类行为识别系统

## 1. 系统原理

### 1.1 毫米波雷达基础
毫米波雷达工作在30-300GHz频段，具有以下特点：
- **高分辨率**：可以精确捕捉人体微小运动
- **全天候工作**：不受光照、天气影响
- **隐私保护**：不采集个人隐私信息
- **穿透性强**：可以穿透非金属材料

### 1.2 信号处理流程
1. **原始数据采集**
   - I/Q信号采样
   - 距离-多普勒处理
   - 角度估计

2. **数据预处理**
   - 距离-速度二维FFT
   - CFAR检测
   - 点云聚类
   - 特征提取

3. **特征提取**
   - 空间特征：点云分布、密度
   - 时间特征：速度、加速度
   - 频域特征：多普勒频谱

### 1.3 多普勒效应
多普勒效应用于测量目标的径向速度：
- f_d = 2v/λ
  - f_d：多普勒频移
  - v：径向速度
  - λ：雷达波长

### 1.4 点云处理
1. **点云生成**
   - 通过角度估计算法获得目标的三维坐标
   - 每个点包含位置(x,y,z)和速度信息

2. **点云聚类**
   - DBSCAN聚类算法
   - 提取人体运动的关键点
