# RadHAR 雷达人类活动识别数据集分析

## 1. 数据集概述

RadHAR（Radar-based Human Activity Recognition）是一个基于雷达的人类活动识别数据集。该数据集使用毫米波雷达采集了多种人类日常活动的数据，用于开发和评估人类活动识别算法。

## 2. 数据集结构

### 2.1 动作类型
数据集包含5种基本动作类型：
- 拳击（Boxing）：26个样本
- 开合跳（Jack）：28个样本
- 跳跃（Jump）：26个样本
- 深蹲（Squats）：29个样本
- 行走（Walk）：33个样本

**总样本数**：142个

### 2.2 数据维度
每个数据点包含12个特征维度：

| 特征类别 | 特征名称 | 描述 |
|---------|---------|------|
| 时间信息 | seq | 序列号 |
|         | secs | 秒数 |
|         | nsecs | 纳秒数 |
| 空间信息 | x | X坐标 |
|         | y | Y坐标 |
|         | z | Z坐标 |
|         | range | 距离 |
| 运动信息 | velocity | 速度 |
|         | doppler_bin | 多普勒频率区间 |
| 其他特征 | point_id | 点的唯一标识符 |
|         | bearing | 方位角 |
|         | intensity | 信号强度 |

### 2.3 样本大小
- 平均每个样本包含：19,460个点
- 数据集总点数：2,763,308个点

## 3. 数据预处理

### 3.1 预处理步骤
1. **数据清洗**
   - 异常值去除
   - 信号降噪
   - 数据规范化

2. **特征提取**
   - 空间特征：位置坐标和距离信息
   - 时间特征：速度和多普勒信息
   - 信号特征：强度和方位信息

### 3.2 可视化方法
1. **点云图**
   - 显示动作的三维空间分布
   - 使用颜色表示信号强度
   - 直观展示动作的空间特征

2. **多普勒频谱图**
   - 展示动作的时频特征
   - 反映动作的速度变化
   - 显示信号强度分布

## 4. 数据集特点

## 8. 实时识别系统

我们开发了一个高度交互式的实时人类活动识别系统，具有以下特点：

### 8.1 系统架构

1. **前端界面**
   - 基于Dash和Plotly的现代化Web界面
   - 深色主题设计，提供专业的视觉体验
   - 响应式布局，适应不同屏幕尺寸

2. **实时可视化**
   - 3D点云动态渲染
     * 实时显示雷达捕捉的点云数据
     * 使用颜色映射表示信号强度
     * 支持交互式旋转和缩放
   
   - 多普勒频谱实时分析
     * 动态更新的频谱热力图
     * 显示速度-时间特征
     * 高分辨率频率分析

3. **动作识别**
   - 基于LSTM的深度学习模型
   - 实时预测当前动作类型
   - 集成决策机制，提高识别稳定性

### 8.2 特色功能

1. **高性能渲染**
   - 优化的3D点云渲染引擎
   - 智能降采样算法
   - 流畅的60FPS显示效果

2. **智能分析**
   - 实时特征提取
   - 动作轨迹追踪
   - 置信度评估

3. **用户交互**
   - 直观的3D场景控制
   - 实时数据统计
   - 动作识别结果即时显示
