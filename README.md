# RadHAR - Human Activity Recognition with Radar Data

This project implements human activity recognition using both millimeter wave and ultra-wideband radar data. The system processes radar data to extract point cloud features and generates 3D visualizations for activity analysis.

## Supported Data Types
- Millimeter Wave Radar (Original)
- Ultra-Wideband Radar (New)

## Key Features
- Point cloud feature extraction
- 3D Doppler spectrograms
- Activity classification
- Multi-radar support

## Directory Structure
- RadHAR/
  - code/ - Main processing scripts
  - data/ - Data preprocessing and utilities
  - docs/ - Documentation and theory
